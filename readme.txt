== Description ==

iMasters WP Dashboard Widget creates a new widget in your WordPress dashboard and shows the iMasters feed. Stay up to date to iMasters.

== Installation ==

How to install iMasters WP Dashboard Widget

e.g.

1. Upload `imasters-wp-dashboard-widget` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use the plugin through the 'iMasters WP Dashboard Widget' menu in WordPress
